Simple project for wrapping a subset of the functionality provided by Fog Creek's Fogbugz XML API with C++.

This code depends on libcurl. (http://curl.haxx.se/libcurl/)

References:
- http://fogbugz.stackexchange.com/fogbugz-xml-api for a full description of the "api"


A test app is provided in the /Test directory.
